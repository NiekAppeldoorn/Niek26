Minecraft

Welkom op de informatie pagina over de wel bekende game, Minecraft.

Minecraft is een game die bestaat uit kubusen. In dit spel is er geen doel om naar toe te werken, het is daarom een zo genoemd sandbox game.

Ondanks dat er geeen doel is, kun je de game wel uitspelen, je leest hier meer over onder het tabblad: Geschiedenis.

Op de website vind je de basic infromatie over minecraft, op de website kun je helaas geen in-game informatie vinden.

Op deze pagina kun je onder andere informatie vinden over: Minecraft zelf, de geschiedenis, en over mij account.

Ik hoop dat deze website nuttig is voor u, veel plezier!
